# Centralized Clinic Token Appointment System

A SaaS platform for clinic appointment management.

## Project Structure

- `backend/`: Node.js Express API with PostgreSQL & Prisma.
- `frontend/`: Next.js Dashboard Portal (Shadcn UI + TypeScript).

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Set up `.env`
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
